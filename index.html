Sawss

An advanced CSS framework

Semantic AWare Style Sheet

Sawss is an intuitive front-end framework for HTML5 development.

Download GitHub

Features

Built as a lightweight and modular HTML5 layout methodology.

Sawss is an extension of CSS3, adding contextual rules, variables, mixins, selector inheritance & more.

Semantic

Sawss makes it easy for you to scaffold HTML5.

Layout

Avoids presentational CSS classes in the markup.

Dynamic

Provides layout for both portrait & landscape.

Design

The beauty of CSS design. Sawss is pure, minimal and transparent.
